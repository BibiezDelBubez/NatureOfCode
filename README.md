# NatureOfCode

a mockery of natural phenomena, or as we call it in Italian: Specchio Riflesso.
Contents so far:

## Genetic algorithm 🧬

- `Hamlet.py`: my pythonesque first attempt to genetic algorithm
- `Susy.py`: Once in a while a popular italian puzzle magazine ("La Settimana Enigmistica") features puzzles which are actually logic or algebra problems in disguise (such as the "Quesito della Susy" column). The "Quesito"" #1021 is a cryptarithm which somehow I deemed as an opportunity to brush up on genetic algorithms. Although, to be honest, it may be not the most efficient way to solve this kind of problem.
